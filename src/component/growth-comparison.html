Polynomial vs Exponential Growth

Why exponential growth eventually wins

The exponential curve uses a small, realistic multiplier: 1.07. It sits below n^2 for a long time, then retakes the lead and keeps widening the gap.

Exponential growth is the math behind compound interest: the next gain is earned on the whole previous result, not just on the original starting amount.

More context

A 7% annual return means each period multiplies the previous balance by 1.07, so after n periods the balance is 1.07^n.

Compound interest with P = $100,000 and r = 7%
A = P ( 1 + r ) n A = 100,000 ( 1 + 0.07 ) n A = 100,000 × 1.07 n

Interest earns more interest. That repeated multiplication is why the green line looks quiet early and then bends upward later.

Polynomial growth like n^2 can look stronger at first because it adds larger absolute chunks early. But it is still built from powers of n. Compound interest is built from repeatedly multiplying by a return factor, and that eventually dominates.
