Sequence of Returns Risk

Why the order of returns matters

Two portfolios can have the same average return and still end very differently when money is being withdrawn.

Bad returns early in retirement shrink the base while withdrawals keep coming out. Good returns later may arrive too late to repair the damage.

More context

This is different from ordinary compounding. Without withdrawals, the order of annual returns does not change the final balance. With withdrawals, the order matters because each withdrawal locks in how much capital remains exposed to the next return.

The calculator uses the same return set in both paths. One path puts the bad years first; the other puts those same bad years last.
