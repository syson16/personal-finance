Drawdown Recovery

Why losses are harder to recover from

Losses and gains are asymmetric. A 50% loss does not need a 50% gain to recover; it needs a 100% gain.

The deeper the drawdown, the more return you need just to get back to even.

More context

Recovery is measured from the smaller post-loss balance. If $100 falls to $50, a $50 gain is now a 100% return because the base is only $50.

This is why avoiding large losses matters. Compound interest can help rebuild capital, but the recovery timeline gets longer as the drawdown gets deeper.
