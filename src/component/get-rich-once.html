Get Rich Once

Why enough capital changes the game

You only need to get rich once. After that, moderate returns on enough capital can produce enough money without heroic assumptions.

A normal return on a large base can produce more dollars than an extraordinary return on a small base.

More context

Before the target, growth matters because the capital base is still small. After the target, the same moderate percentage return produces a much larger dollar amount.

The point is not that 5% is magic. The point is that enough capital turns ordinary returns into meaningful annual income.
